# AR-project-Crane-Safety

AR Crane Safety Viewer is a Unity-based Android app using AR Foundation and ARCore to teach crane geometry through interactive 3D and AR visualization. Users can explore, rotate, label, and capture crane models, improving pre-lift understanding, safety awareness, and learning efficiency.
